Getting Started

This tool converts BIDS raw data into a structured DataLad repository, suitable for collaborative neural data management at MRI-Lab Graz.

  • Source: Select the folder containing subject directories (sub-01, etc.)
  • Destination: Choose where to create the new DataLad dataset.
  • SSH: Enable the remote toggle to push directly to a central server.

For more details, visit the Documentation.
